暗号技術特論期末レポート

注意事項:

•期末レポートは、従来の期末試験に代わるレポート課題です。解答をレポートの形でまとめ、指定された締め切り日までにmanabaから提出してください。

•手書きノートを「スキャンアプリ」などで電子化したファイルでも構いませんが、読みやすいようにスキャンするとともに、最終的にはpdfファイルとして提出してください(解答ファイルが複数になる場合はzipファイルとして圧縮して1ファイルとして提出するようお願いします)。

•日本語または英語で解答してください。

•必ず問題番号を書いてから解答を記述してください。レポートに問題文を再掲する必要はありません。

•これは自宅試験(take-home exam)に相当するため自分の力のみで解いてください(自分以外の人に公開する，相談するなどして協力を得ないでください，不正行為に相当します)。

•解答作成にあたって、講義資料(演習問題含む)や外部の資料，インターネットの情報などの参照は自由です。しかし外部資料等を参照してその事実や定理などを使う場合は必ず出典を明記してください(出典を明記しても減点にはなりません)。

•様々な資料の参照が自由な形式なため，ある程度の差がつくよう，また誰にとっても何かしら解ける問題が含まれているよう，問題数は通常の期末試験よりかなり多めに設定しています。そのためAを得るためには全ての問題を解かなければならないということはありません。また解けた問題数が少なくとも平均点調整により単位を得られる可能性があります。

2つのコップを組み合わせてLの中に1リットルの水を入れるゲームを行うとする(n, mは正の整数)。

(a)例えばn= 3, m= 5であれば，nのコップで2回，Lに水を入れ，Lからmのコップで1回，水をすくい出せばLの中に1リットルの水が残り成功。

(b)例えばn= 11, m= 7であれば，nのコップで2回，Lに水を入れ，Lからmのコップで3回，水をすくい出せばLの中に1リットルの水が残り成功。

一方で例えばゲームの主催者からn=m= 3のコップを与えられた場合，このゲームには多分成功できそうにない。

ゲームの主催者の不正を防ぐために，n, mを与えられ，ゲームに成功できるか否か判定する方法(原理の説明が正しければ厳密でなくてもよい)とその正当性を説明せよ。

gcd(m,n)!=1のとき、成功できない

gcd(m,n) = aに対して、k1*m + k2*n = 1と仮定する

mk1 + nk2 = a*(k1*m/a + k2*n/a)

mk1 + nk2 = a*(k1*m/a + k2*n/a) >= a

したがって、1リットルの水を入れられない

11.negligible関数は講義で紹介した定義以外にも複数の等価な定義が知られている。例えば「どのような整数i >0に対してもlimn→∞nif(n) = 0のときf(n)はnegligible関数である」という定義も存在している。このときf(n) =1nlog2(log2(n))はnegligible関数か否か理由とともに説明せよ。

$F(n) = n^{i} * \frac{1}{n^{\log_{2}{(\log_{2}{n})} }} = n^{\log_{2}{2^i}-\log_{2}{(\log_{2}{n})}} = n^{i + \log_{2}{(\log_{n}{2})}} = \frac{1}{n^{-\log_{2}{(\log_{n}{2})-i}}}$

$\lim_{n \to \infty}{n^{-\log_{2}{(\log_{n}{2})-i}}} = +\infty $
$\lim_{n \to \infty}F(n) = 0 $

したがって、f(n)はnegligible関数である
